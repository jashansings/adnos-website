# ADNOS Studios Website

Portfolio website for ADNOS Studios, built with React, Vite, TypeScript, Tailwind CSS, and Motion.

## Run Locally

Prerequisite: Node.js

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

No backend or API keys are required.
